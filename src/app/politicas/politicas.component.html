Políticas de Privacidad

Esta aplicación es una aplicación interna de uso privado, creada exclusivamente para la gestión técnica y administrativa de accesos a APIs proporcionadas por Meta Platforms Inc., incluyendo, entre otras, APIs de WhatsApp, Instagram y Facebook.

La aplicación no es de acceso público, no está dirigida a usuarios finales y no ofrece servicios directamente a personas naturales.

La aplicación no recopila, no almacena ni procesa datos personales de usuarios, clientes o terceros.

No se realizan procesos de autenticación de usuarios finales, no se accede a mensajes, contactos, perfiles, números telefónicos ni información privada.

El uso de las APIs de Meta se limita a funciones técnicas internas previamente autorizadas, y se ejecuta únicamente por administradores del negocio con permisos controlados.

La aplicación puede generar registros técnicos internos (logs) necesarios para el funcionamiento del sistema, los cuales no contienen información personal identificable.

En consecuencia, esta aplicación no realiza tratamiento de datos personales en los términos definidos por la legislación aplicable.

Esta aplicación no recopila ni almacena datos personales de usuarios.

En consecuencia, no existe información personal que pueda ser objeto de solicitudes de eliminación.

En caso de futuras modificaciones que impliquen tratamiento de datos personales, esta política será actualizada previamente y se habilitarán los mecanismos correspondientes.
